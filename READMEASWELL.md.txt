
## Setup

1. Clone the repository
2. Open any HTML file in a web browser:
   - `amazon.html` - Main product page
   - `orders.html` - Order history
   - `checkout.html` - Checkout process
   - `tracking.html` - Order tracking

## Dependencies

- Google Fonts (Roboto)
- Local images for icons and logos
- Modern browser with JavaScript support

## Technical Details

- **Responsive Design**: Uses viewport meta tag and responsive CSS
- **Styling**: CSS Grid and Flexbox layouts
- **Font**: Roboto from Google Fonts (400, 500, 700 weights)
- **JavaScript**: Modular scripts for each page's functionality

## Notes

- Cart quantities are hardcoded in the examples (would be dynamic in full implementation)
- Requires all asset files (images, CSS, JS) to be in correct directories
- Designed as a frontend demonstration (no backend integration)
